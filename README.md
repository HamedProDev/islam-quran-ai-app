# 📱 Overview

Islam Quran AI is an intelligent Islamic platform that provides Quran reading, AI-powered Islamic assistance, prayer times, Quran tafsir, Islamic learning resources, and many advanced features designed specifically for Muslims worldwide.

The application aims to make authentic Islamic knowledge accessible through modern technology while maintaining respect for Islamic values and teachings.

# 🚀 Installation

## Requirements

* Flutter SDK
* Android Studio
* Dart SDK
* Android Device or Emulator

## Clone Repository

```bash
git clone https://github.com/yourusername/islam-quran-ai.git
```

```bash
cd islam-quran-ai
```

## Install Dependencies

```bash
flutter pub get
```

## Run App

```bash
flutter run
```
